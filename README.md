## ⚡ ¿Por qué OpenPDF?
Olvídate de pagar costosas suscripciones mensuales o lidiar con herramientas en línea que comprometen la privacidad de tus documentos. **OpenPDF** es una aplicación de escritorio nativa diseñada para ponerte al control de tus flujos de trabajo de documentos desde un entorno seguro y con una interfaz moderna e impecable que se adapta a ti.

Desde firmar contratos hasta combinar reportes masivos y proteger archivos confidenciales, **OpenPDF** ha sido optimizado y refinado hasta el más mínimo detalle para asegurar un rendimiento excepcional.

---

## ✨ Características Principales

### 🔄 Multi-Conversión Universal
* **A tu manera:** Convierte rápidamente documentos de Word (`.docx`), imágenes (`.png`, `.jpg`) y textos en archivos PDF profesionales.
* **Fusión Inteligente:** Une múltiples archivos en un único documento, perfecto para portafolios, proyecciones y entregas de reportes exhaustivos.

### 🎨 Diseño y UX de Nueva Generación
* **Modo Oscuro Integrado:** Disfruta de un entorno relajante y profesional para trabajar a deshoras, con un solo clic.
* **Archivos Recientes Cerca de Ti:** Accede a tus documentos más importantes desde el menú principal. Todo sincronizado en el momento.
* **Estadísticas de Documento:** Interfaz interactiva para monitorizar las dimensiones de páginas, el peso del archivo y los cambios efectuados al instante.

### 🛡️ Seguridad Avanzada
* **Protección con Contraseña:** Cifra fácilmente tus archivos para asegurar que la información sólo caiga en las manos correctas.
* **Censura Visual Interactiva:** Arrastra el ratón directamente sobre datos ultra-secretos para redactarlos de forma inquebrantable desde el propio lienzo.

## 🖌️ Funcionalidades de Edición Nativas y Profesionales

OpenPDF te regala una amplia experiencia de edición WYSIWYG interactiva que permite manipular visualmente el PDF como si tuvieras papel y lápiz en mano.

* **Dibujo Libre & Anotaciones:** Toma notas con nuestro motor de trazo libre personalizable o añade **Notas Adhesivas** arrastrables ideales para marcar revisiones.
* **Gestión de Páginas Completa:** Reorganiza el flujo de un texto arrastrando y soltando visualmente miniaturas, rota vistas o recorta imperfecciones rápidamente.
* **Herramientas de Inserción:** Incrusta numeración de páginas impecable, marcas de agua personalizadas, imágenes, hipervínculos funcionales y formas geométricas.
* **Atajos de Teclado & Historial:** Muévete a la velocidad de la luz. Guarda (`Ctrl+S`), Imprime nativamente (`Ctrl+P`), e interactúa sin temor a equivocarte gracias a nuestro poderoso motor de **Deshacer (`Ctrl+Z`) y Rehacer (`Ctrl+Y`)**. Todo está respaldado.

## 📥 Instalación

**OpenPDF** está paquetizado como un software listo que no requiere configuraciones extras para Windows.

### Para Usuarios 
1. Dirígete a la carpeta `dist/`.
2. Ejecuta el archivo `OpenPDF Setup 1.0.0.exe`.
3. Sigue el asistente de instalación con un-clic y comienza a dominar tus PDFs inmediatamente.

### Para Desarrolladores (Entorno Dev)
1. Clona el repositorio.
2. Instala las dependencias necesarias:
   ```bash
   npm install
   ```
3. Ejecuta el entorno de pruebas local:
   ```bash
   npm run dev
   ```

## 🛠️ Tecnologías

Construido para ser duradero y eficiente, OpenPDF combina tecnología web de punta con integraciones de sistema de bajo nivel:
- **[Electron](https://www.electronjs.org/)**: Como núcleo robusto ejecutando una app nativa cross-platform.
- **[pdf-lib](https://pdf-lib.js.org/)**: Procesamiento de PDF limpio, rápido y altamente confiable.
- **[pdf.js](https://mozilla.github.io/pdf.js/)**: Renderización en Canvas con una fidelidad superior.
- **HTML5/CSS3/Vanilla JS**: Para asegurar la aplicación más ligera de PDF sin dependencias que ralenticen tu PC.
